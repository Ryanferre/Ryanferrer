certificados

  • CSS FLEX-BOX

    Esse curso aborda conceitos básicos do CSS, mas o seu foco é o Flexbox para criação de projetos, deixando-os de acordo com o designer proposto. Foram abordados assuntos como EM QUAIS CONTEXTOS UTILIZAR O FLEXBOX, UTILIZAÇÃO EM INTERFACES RESPONSIVAS, DOMÍNIO DOS FLEX CONTAINERS E FLEX ITEMS.

  • Git e Github

    Esse curso aborda conceitos e métodos de versionamento utilizando Git para controlar versões do software e códigos, e GitHub para armazenamento remoto. São abordados temas como: USO DE SSH, PROTEÇÃO DE BRANCHES E CONTROLE DE ACESSOS, COMANDOS GIT, PULL REQUESTS, CI/CD E INTEGRAÇÃO COM GITHUB ACTIONS.

  • Git e Github(practical)

    Esse curso é um complemento do anterior, porém com foco mais aprofundado em projetos e resolução de problemas no Git. Foram abordados conteúdos como: ENTENDENDO COMO TERMINAL WINDOWS/MAC FUNCIONA, CRIAÇÃO DE REPOSITÓRIO LOCAL E REMOTO (GITHUB), SOLUÇÃO DE CONFLITOS, MESCLAGENS E GESTÃO DE ORGANIZAÇÕES, FORKS, PULL REQUESTS E ADIÇÃO DE MEMBROS ÀS ORGANIZAÇÕES.

  • JavaScript(projects)

    Esse curso aborda conceitos importantes utilizando JavaScript. Foi colocado em prática o conhecimento adquirido nas aulas, com projetos que envolvem: ORIENTAÇÃO A OBJETOS, ESTRUTURAS DE CONTROLE (SWITCH, IF/ELSE, LOOPs, ETC), MANIPULAÇÃO DE ARRAYS, HTML E CSS, IMPORTAÇÃO/EXPORTAÇÃO DE MÓDULOS E TIPOS DE DADOS.

  • javascript(basico ao avancado)

    Esse curso é um complemento do anterior, porém mais avançado e com conteúdos aprofundados. Foram abordados temas como: ESTRUTURA DE PROGRAMAÇÃO, FUNÇÕES E ARROW FUNCTION, CLOSURE, RECURSION, ESCOPO ANINHADO, MANIPULAÇÃO DE ARRAYS E ORIENTAÇÃO A OBJETOS, DEBUGS, ERROS, PROGRAMAÇÃO ASSÍNCRONA, API REST/RESTful, VERBOS HTTP, TESTES E ROTAS.

  • react, bootstrap e redux

    Esse curso é um complemento do anterior, porém mais avançado e com conteúdos mais aprofundados. Foram abordados temas como: CONSUMO DE APIs EXTERNAS, MANIPULAÇÃO E APRESENTAÇÃO DOS DADOS DE API, MANIPULAÇÃO DE ESTADO E USO DE COMPONENTES, ESTILIZAÇÃO COM TAILWINDCSS, TYPESCRIPT PARA CÓDIGOS TIPADOS E HOOKS DE ESTADO.

  • tailwincss(do basico ao avancado)

    Este curso de Tailwind vai do básico ao avançado. Ensina a ler a documentação, preparar o ambiente e utilizar classes utilitárias ligadas ao CSS. Foram abordados conceitos como: PSEUDOCLASSES, EVENTOS, FLEXBOX E GRID, BREAKPOINTS, ESPAÇAMENTO, TIPOGRAFIA E BACKGROUND, ESTILIZAÇÃO DE BORDAS E INTERATIVIDADE COM TAILWIND.

  • typescrit(projects)

    Esse curso de TypeScript vai do básico ao avançado. Ensina a ler a documentação e preparar o ambiente. Foram abordados conceitos como: O QUE É TYPESCRIPT, FUNCTION RETURN E VOID, TIPOS, OBJETOS, ARRAYS E CONFIGURAÇÃO DO WATCH MODE, ROOTDIR, OUTDIR, PUBLIC/PRIVATE E INTERFACES, CLASSES, UNUSED LOCALS E UNUSED PARAMETERS.

  • typescript

    Esse curso de TypeScript com React foca na prática de conceitos importantes usados por profissionais. Foram abordados temas como: INTEGRAR E CRIAR APLICAÇÕES COM REACT E TYPESCRIPT, USO DE HOOKS COM TYPESCRIPT (useReducer, Context API, useEffect...), FUNDAMENTOS DO TYPESCRIPT E DO REACT JS, REQUISIÇÕES HTTP E PRINCIPAIS RECURSOS DO TYPESCRIPT NO REACT.
